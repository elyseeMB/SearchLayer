# SearchLayer - Architecture de Feature Agnostique

Projet de démonstration montrant comment architecturer une feature de recherche agnostique, séparée du domaine métier, en utilisant **AdonisJS**, **Inertia** et **Svelte**.

## Objectif

Illustrer les meilleures pratiques pour créer une couche de d'abstraction réutilisable et découplée de la logique métier.

## Stack Technique

- **Backend**: AdonisJS
- **Frontend**: Svelte + Inertia
- **Architecture**: Feature agnostique + Domain-driven design

## Concepts Démontrés

- Séparation des responsabilités
- Architecture en couches
- Design pattern agnostique
- Réutilisabilité des composants

## Démarrage Rapide

```bash
# Installation
npm install

# Base de données
node ace migration:run

# Seeds
node ace db:seed

# Développement
npm run dev
```
